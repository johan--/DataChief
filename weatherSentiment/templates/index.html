About This Site


Twitter Data Collection

Using Twilio, a powerful Twitter API, we are able to sample large amounts of tweets containing the weather keyword at random, and save them by their corresponding location.

Google Maps API

The Google Maps visualizations are generated through use of the Heat map Data layer, which enables us to translate sentiment data to a color coated point depending on sentiment score.

Django Backend

Using Django's back end technologies and SQLAlchemy, we run queries on real-time data based on various locations. This makes handling data easy, and tracking changes over time even easier.
